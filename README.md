## Preparation

Need prepare database first, execute [db.sql](db.sql) file for ready database

## Installation

Create `.env` file and add this

```sh
APP_HOST=host
APP_PORT=port
PGHOST=postgres_host
PGUSER=postgres_user
PGDATABASE=postgres_db
PGPASSWORD=postgres_passwrod
PGPORT=5432
```

## Deploying

And run following

```sh
npm run deploy
```
